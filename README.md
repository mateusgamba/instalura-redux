### Instalura with Redux

Project based on the Instagram, developed to learn React


Install nodejs and your dependences
```
npm install
```

Run instalura app
```
java -jar instalura.jar
```

Include data in the database, execute:
```
http://localhost:8080/gera/dados
```

Run the application
```
npm start
```
